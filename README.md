# minio-spring-boot-starter

# 使用方法

### 依赖

```

<dependency>
    <groupId>plus.ojbk</groupId>
    <artifactId>minio-spring-boot-starter</artifactId>
    <version>1.0.0</version>
</dependency>

```

### 配置

```

file:
  minio:
    enable: true
    endpoint: http://172.16.1.240:9000
    bucket: test
    access-key: test
    secret-key: test123456789

```
